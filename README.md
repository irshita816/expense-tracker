# Expense Tracker

A responsive Expense Tracker developed using HTML, CSS, and JavaScript.

## Features

- Add Income
- Add Expense
- Delete Transactions
- Automatic Balance Calculation
- Transaction History
- Local Storage Support
- Responsive User Interface

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## How to Run

1. Download or clone this repository.
2. Open `index.html` in your browser.
3. Start adding your income and expenses.
